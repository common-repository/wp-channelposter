WP ChannelPoster - Import Youtube.com videos from Channels or selected Playlists and publish on your WordPress webblog!

== Description ==

**WP ChannelPoster** is an incredibly fast, simple, yet powerful, method of embedding YouTube videos from your YouTube Channel and Playlist to your self hosted WordPress blogs.

**Key features include:**

* Automatically post your **YouTube channel or playlist** to your wordpress blog
* You can select which playlist or everything in a channel
* Set up **categories based on playlist**.
* Create separate playlist on YouTube fore separate niche content. 
* Post from a YouTube channel to multiple sites and maintain unique content by using playlist. 
* Modify posting templates to include your own content, images, links or banners 
* Use internal WP Cron to post and many videos as you want per post every X hours
* Can be integrated with other tools to amplify your sites traffic
* Automatically SEO optimized

== Installation ==

1. Upload the WP ChannelPoster Plugin to your blog (in the 'wp-content/plugins' directory) (be sure to upload the "wp_channelposter" directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress. There are user instructions and video training available from the plugin setting page.
1. Use the wp-admin->Settings->WP ChannelPoster screen to configure the plugin.
1. **You would need a Youtube.com API key to begin importing**.

== Frequently Asked Questions ==

= Can we Import Youtube.com Videos from multiple Playlists =

Yes! You can import videos from multiple playlist within the selected Channel.

= I cant find my Youtube.com User Name =

Youtube moved away from usernames, to channels a while back, so if you have an account which was created after the shift, you would want to use the channel ID or the Playlist ID options.
